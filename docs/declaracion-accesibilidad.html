Declaración de Accesibilidad

El Técnico TI del Pueblo se ha comprometido a hacer accesible su sitio web de conformidad con el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público (que traspone la Directiva (UE) 2016/2102 del Parlamento Europeo y del Consejo).

>_situacion_de_cumplimiento

Este sitio web es **plenamente conforme** con la norma UNE-EN 301549:2022 y las pautas WCAG 2.1 en su nivel AA. Todo el portal estático ha sido diseñado bajo los siguientes criterios de accesibilidad:

  • Estructura Semántica: Jerarquía correcta de títulos (h1, h2, h3) y etiquetas HTML5 semánticas (header, main, nav, section, footer) para facilitar la lectura mediante lectores de pantalla.
  • Contraste de Color: Aseguramos un contraste mínimo de color de 4.5:1 para cumplir con las guías de legibilidad, adaptando la paleta de colores retro-tech.
  • Navegación por Teclado: Todos los elementos interactivos (botones, enlaces, menú hamburguesa) disponen de estados `:focus` altamente visibles y son navegables mediante la tecla Tabulación. Se incluye un enlace de acceso rápido ("skip-link").
  • Etiquetas Aria y Descriptivas: Elementos decorativos disponen de etiquetas descriptivas alternativas (`alt` y `aria-label`) para imágenes y carruseles.

>_contacto_y_sugerencias

Si experimenta cualquier tipo de problema de visualización, navegación o accesibilidad en este sitio, o desea realizarnos cualquier consulta o sugerencia de mejora, puede ponerse en contacto directo a través de la dirección de correo electrónico michelmacias.it@gmail.com indicando en el asunto "Accesibilidad Web".
